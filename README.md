# Grocery-Price-Comparison
Multi-Store Price Aggregator with Delivery-Time Optimization Built with Streamlit + Python | Deployable via ngrok on Colab

# Overview
This project compares prices and delivery times for daily essential items across multiple grocery delivery services:
Blinkit
Zepto
Instamart
JioMart
Users can search for multiple items at once, and the system returns:
Best store for each product
Total cart cost per store
Average delivery time per store
Cheapest overall store for the entire list
Clean card-based UI

# Features
✔ Multi-item search
✔ Per-item lowest price
✔ Overall cheapest store calculation
✔ Delivery time included
✔ Professional UI (cards + non-scroll layout)
✔ Fully working on Google Colab
✔ ngrok tunnel for public deployment
✔ Beginner friendly + resume ready

# Tech Stack
Python 3
Streamlit for UI
Pandas for dataset handling
pyngrok for tunneling in Colab
