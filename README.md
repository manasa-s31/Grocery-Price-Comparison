# Grocery-Price-Comparison

# Overview
Multi-store grocery price and delivery-time aggregator built with Streamlit and Python.  
Compares prices across multiple grocery providers and provides recommendations for cheapest options overall and per item.

# Grocery providers
1. Blinkit
2. Zepto
3. Instamart
4. JioMart
   
Users can search for multiple items at once, and the system returns:
1. Best store for each product
2. Total cart cost per store
3. Average delivery time per store
4. Cheapest overall store for the entire list
5. Clean card-based UI

# Features
✔ Multi-item search
✔ Per-item lowest price
✔ Overall cheapest store calculation
✔ Delivery time included
✔ Professional UI (cards + non-scroll layout)
✔ Fully working on Google Colab
✔ ngrok tunnel for public deployment
✔ Beginner friendly + resume ready

# Tech Stack
1. Python
2. Streamlit for UI
3. Pandas for dataset handling
4. pyngrok for tunneling in Colab
